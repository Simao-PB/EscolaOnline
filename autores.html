Sobre o Projeto

Este projeto foi desenvolvido como parte da avaliação da Unidade Curricular de Composição de Interfaces Multiplataforma, do curso de Informática Web Movél e na Nuvem da Universidade da Beira Interior.

O objetivo foi criar um site responsivo para uma escola com foco em b-learning, aplicando os conhecimentos adquiridos sobre design de interfaces, acessibilidade web e técnicas de responsive design.

Utilizamos tecnologias modernas como HTML5, CSS3 e Bootstrap para desenvolver uma interface intuitiva, acessível e adaptável a diferentes dispositivos.
